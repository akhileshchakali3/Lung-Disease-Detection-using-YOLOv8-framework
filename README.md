# A Class Balanced YOLOv8 Framework for Automated Lung Disease Detection in Chest X-ray Images

## Overview

This project is an AI-powered web application that detects lung diseases from Chest X-ray images using a Class Balanced YOLOv8 model. It provides disease detection, AI-generated medical reports, and an interactive chatbot through a simple web interface.

## Features

- Chest X-ray image upload
- Lung disease detection using YOLOv8
- AI-generated medical report
- PDF report generation
- AI chatbot using Google Gemini
- Simple and responsive web interface

## Technologies Used

- Python
- FastAPI
- Ultralytics YOLOv8
- OpenCV
- HTML
- CSS
- JavaScript
- Google Gemini API
